・本ソフトウェアについて

本ソフトウェアは、平成25年度 総務省「オープンデータ実証実験 自治体行政情報実証事業（参考 http://www.soumu.go.jp/main_content/000262475.pdf ）」において実証に用いた「自治体行政情報流通連携基盤システム（以下、本ツール）」のうち、インディゴ（株）シームレス空間基盤研究開発センター（ラボチーム）が開発した一部ソースコードについて、類似ツール構築に際しての「サンプルコード」として参照されることを想定し「MITライセンス」の元で公開するものです。
なお、本リポジトリには、本ツール構築に必要な既存オープンソース・ソフトウェア等は含んでおりません。本ツールの構築については、doc/構築手順.pdf を参照してください。


・ライセンスについて

ライセンス内容については下記ディレクトリ構成に示す LICENSE.txt を参照のこと。
